## ğŸ¯ Overview

**KoreaMemory** is an Endless Runner game inspired by the cooperation between Vietnam and South Korea during the COVID-19 pandemic. The game integrates cultural representation and pandemic-related elements to convey a meaningful message about solidarity, resilience, and lasting friendship between the two nations.

Developed for the Korea Game Week Competition, this project demonstrates how interactive media can communicate important social values while providing engaging gameplay.

## ğŸš€ Getting Started

### Prerequisites

- Unity 2020.3 or higher
- Visual Studio or compatible IDE
- Basic understanding of Unity game development

### Installation

```bash
# Clone the repository
git clone https://github.com/NinITWB/KoreaMemory.git

# Navigate to the project directory
cd KoreaMemory

# Open the project in Unity
# File -> Open Project -> Select KoreaMemory folder
```

### How to Play

1. **Start Game**: Launch from the main menu
2. **Control Character**: Use arrow keys or touch controls to move
3. **Collect Items**: Gather masks, vaccines, and lucky boxes
4. **Avoid Obstacles**: Dodge viruses to maintain health
5. **Survive**: Run as far as possible to achieve high scores
